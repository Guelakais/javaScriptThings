#basicGitCommands
git init # transforms the folder, you're actual in in a get repo
git commit -m "comment" //how to make a commit
git branch -M name //adds all maked commits to a branch
git add file //adds a file to the next commit
git remote add origin link //adds a origin to your git repo, were the next push order gets in
git push -u origin name //pushes the called branch to your origin
git pull
